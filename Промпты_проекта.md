# Промпты проекта

Проект: Telegram-бот "Ассистент по встречам"  
Статус: MVP завершен и готов к эксплуатации  
Дата подготовки документа: 12.07.2026

Этот документ собирает рабочие промпты, которые можно использовать для поддержки,
проверки, доработок и повторного запуска отдельных этапов проекта.

## Как пользоваться

1. Для новой задачи сначала используйте базовый промпт из раздела 1.
2. Для работы по этапам используйте промпты из раздела 2.
3. Для эксплуатации, проверки production и UAT используйте раздел 3.
4. Для будущих доработок v1.1/v2 используйте раздел 4.
5. Не вставляйте реальные секреты, токены, пароли, refresh token и значения `.env`
   в промпты. Если нужно проверить `.env`, просите выводить только
   `configured true/false`.
